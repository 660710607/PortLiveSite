Telecom Churn Analysis

Python SQL Server Power BI Machine Learning

Goal

Developed a complete analytics project to understand and predict customer churn. Built an ETL pipeline in SQL Server to clean and transform customer data. Designed a Power BI dashboard to visualize key metrics such as churn rate, customer segments, and revenue loss. Implemented predictive modeling to identify high-risk customers and support proactive retention strategies.

การศึกษาข้อมูลลูกค้าเพื่อดูว่าลูกค้าคนไหนมีแนวโน้มจะเลิกใช้บริการ (churn) , ช่วยให้เข้าใจพฤติกรรมลูกค้า และปัจจัยเสี่ยงต่างๆ , และหาปัจจัยที่ทำให้เกิด churn เพื่อช่วยให้บริษัทสามารถวางแผนกลยุทธ์รักษาลูกค้าได้ดีขึ้น

Tools & Techniques

  • Database & ETL: ใช้ SQL Server สำหรับจัดเก็บและแปลงข้อมูลจากไฟล์ CSV เข้าสู่ฐานข้อมูล
  • Data Cleaning: ตรวจสอบค่าว่าง, จัดกลุ่มข้อมูล และจัดการค่า inconsistent ก่อนวิเคราะห์
  • Power BI: สร้าง Dashboard ที่แสดงภาพรวมเชิงข้อมูล เช่น Churn Rate, จำนวนลูกค้าทั้งหมด, การแบ่งกลุ่มตามเพศ อายุ หรือประเภทบริการ
  • Predictive Modeling: ระบบอาจรวมการทำนายลูกค้าที่มีแนวโน้มจะ churn ด้วย Machine Learning โดยใช้ algorithm Random Forest และอื่นๆ และแสดงผลลัพธ์บน Dashboard

Workflow

  • Import and Load Data: นำข้อมูลลูกค้าจากไฟล์ .csv เข้าไปในฐานข้อมูล SQL Server
  • ETL & Cleaning: ตรวจสอบข้อมูลที่ขาดหาย และย้ายข้อมูลไปตาราง production สำหรับใช้งาน
  • Data Transformation: สร้างตารางช่วย (auxiliary tables) เพื่อจัดกลุ่มอายุ, ระยะเวลาการเป็นลูกค้า, และระดับค่าบริการ
  • Dashboard: สร้าง Power BI Dashboard ที่มี metrics หลัก
  • Prediction: ใช้โมเดล Machine Learning เพื่อทำนายลูกค้าที่อาจ churn ในอนาคต และเชื่อมผลลัพธ์กับ Power BI

Framework

  • Database / ETL : SQL Server , SSMS
  • Data Transformation : Power BI , Excel
  • Machine Learning : sklearn.ensemble.RandomForestClassifier
  • Model Evaluation : sklearn.metrics.classification_report, sklearn.metrics.confusion_matrix
  • Data Handling : Python (pandas , numpy , sklearn.preprocessing.LabelEncoder)
  • Visualization : Power BI , matplotlib , seaborn
  • API : pandas , numpy , scikit-learn , matplotlib / seaborn , joblib

วิธีการแบบคร่าวๆ

  • 1. เชื่อม SQL Server และสร้าง Database (ETL)

    • 1.1 เตรียม SQL Server และ Database
    - dataset : https://pivotalstats.com/wp-content/uploads/2024/08/Data-Resources.zip
    - SQL Server และ SQL Server Management Studio (SSMS)
    - Server: ชื่อ SQL Server (เช่น localhost\SQLEXPRESS)
    - Database เช่น db_Churn
    1.2 สร้าง Table ชั่วคราวสำหรับโหลดข้อมูล (stg_table)
    - นำข้อมูล CSV โหลดเข้ามาใน stg_Churn
    - ก่อนนำเข้าข้อมูลจริง , สร้าง stg_Churn เพื่อเช็คความถูกต้องของข้อมูลก่อน
    1.3 ทำความสะอาดข้อมูล (Data Cleaning / Transformation)
    - แทนค่าที่เป็น NULL ด้วยค่า default หรือ “Unknown”
    - แก้ค่าที่ไม่ถูกต้อง (เช่นถ้า Total_Charge น้อยกว่า 0 ให้เป็น 0)

  • 2. Load ข้อมูลสู่ Production Table และ Data Transformation

    • 2.1 สร้าง Production Table โดยนำข้อมูลจาก stg_Churn

    - Clean / Transform Data เพิ่มเติม
    2.2 สร้าง Column ใหม่สำหรับวิเคราะห์
    - Age Group (แยกอายุ เช่น ใน 20 35 50 ปี)
    - Tenure Group (อายุการใช้งาน เช่น 6 เดือน 12 เดือน 1-2 ปี)
    2.3 Validate Production Table - ตรวจสอบว่า ไม่มี NULL หรือค่าผิดปกติ ในคอลัมน์สำคัญ (Customer_ID, Churn_Flag)
